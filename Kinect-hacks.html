Kinect Your Blackboard

About

This is an online blackboard. Using a Microsoft Kinect, this application will track the hand movements of a person. It will then keep track of the coordinates, draw a picture of the coordinates, and then convert this picture to text using Computer Vision. It then displays this text and allows the user to save it, along with a variety of other options. The demo website does now implement the kinect, it is just the layout of the website.
